# Privacy Policy

**Pokemon Auto Chess Live Data Calculator**

*Last updated: November 26, 2025*

## Data Collection

This extension does **not** collect, store, or transmit any personal data.

## Local Storage

The following data is stored locally on your device using browser localStorage:

- Calculator window position
- Your in-game player name (if you choose to enter it)
- Team target preferences

This data never leaves your device and is not accessible to anyone except you.

## Network Activity

This extension makes **zero** network requests. There are:

- No analytics
- No tracking
- No telemetry
- No third-party services
- No data transmission of any kind

## Permissions

| Permission | Why It's Needed |
|------------|-----------------|
| activeTab | Inject calculator overlay on pokemon-auto-chess.com |
| scripting | Read public game state for probability calculations |
| storage | Save your preferences locally |
| Host (pokemon-auto-chess.com) | Extension only runs on this site |
