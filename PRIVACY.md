# Privacy Policy

**Pokemon Auto Chess Live Data Calculator**  
*Last updated: December 7, 2025*

## Data Collection

This extension does **not** collect, store, or transmit any personal data.

## Local Storage

The following data is stored locally on your device using browser localStorage:

- Calculator window position and size
- Your in-game player name (for shop flash alerts)
- Team Tracker target preferences
- Polling speed preference
- Roll Luck / Shop History session data
- EULA acceptance status
- Experimental features unlock status
- Mono-Type Challenge preferences

This data never leaves your device and is not accessible to anyone except you.

## Game State Reading

This extension reads publicly available game state data to provide probability calculations:

- Pokemon on all players' boards, benches, and shops
- Current stage number
- Player level
- Shop contents for roll tracking

This data is processed entirely within your browser and is **never transmitted** anywhere.

## Network Activity

This extension makes **zero** network requests. There are:

- No analytics
- No tracking
- No telemetry
- No third-party services
- No data transmission of any kind

## Permissions

This extension uses **minimal permissions**:

| Permission | Why It's Needed |
|------------|-----------------|
| Content Scripts (pokemon-auto-chess.com) | Extension only runs on this site |

This extension requests **no other permissions**. Specifically:

- ❌ No `host_permissions`
- ❌ No `activeTab`
- ❌ No `storage` API permissions
- ❌ No background scripts
- ❌ No service workers

Local preferences are saved using the browser's built-in localStorage API, which requires no special permissions.
